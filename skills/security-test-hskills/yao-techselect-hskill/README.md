# yao-techselect-hskill

技术选型助手 - 帮助团队进行客观的技术选型决策。

## 功能

- 多维度技术对比
- 行业专家观点整合
- 风险评估和迁移成本分析
- 输出结构化推荐报告

## 使用场景

- 新项目技术栈选择
- 现有系统重构方案评估
- 第三方服务/库的选型
- 云服务供应商对比

## 快速开始

激活 skill 后，描述你的选型需求：

```
帮我对比 React、Vue 和 Svelte，用于中型企业管理后台项目。
团队有 React 经验，但愿意学习新技术。
```
